
# PÁTIO DE AUTOMÓVEIS - LISTAGEM DE ÁREAS

Projeto desenvolvido em PHP na aula do curso Tecnico Em desenvolvimento de sistemas, esse projeta utiliza, HTML, Bootstrap e MYSQL. Sofware para listagem de caros e vendas de carro. 


## Tecnologias

 - [PHP](https://www.php.net/)
  - [Banco de dados](https://www.heidisql.com/)
- [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- [Bootstrap](https://getbootstrap.com/)
- [Mysql](https://www.mysql.com/)
